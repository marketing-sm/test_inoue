# 3問性格診断

3つの質問に答えるだけで、あなたの性格タイプを診断するWebアプリです。

## 特徴

- 質問3問で最大27タイプを判定
- レスポンシブデザイン（スマホ対応）
- ダークモード対応
- サーバー不要・ファイル単体で動作

## 使い方

`index.html` をブラウザで開くだけで動作します。

## 診断の仕組み

| 軸 | 選択肢 |
|---|---|
| Q1：エネルギーの向き | 社交派(E) / 内省派(I) / 柔軟派(M) |
| Q2：問題への対処 | 分析家(T) / 共感型(F) / 直感型(X) |
| Q3：行動スタイル | 計画型(J) / 自由型(P) / 仲間型(S) |

3軸の組み合わせで最大27タイプを判定します。
